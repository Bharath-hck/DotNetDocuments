## C# Operators

Addition Operator `+`

The addition operator + computes the sum of its operands:

``` csharp
// Some comments
    internal class Program
    {
        static void Main()
        {
            Console.WriteLine(5 + 4);       // output: 9
            Console.WriteLine(5 + 4.3);     // output: 9.3
            Console.WriteLine(5.1m + 4.2m); // output: 9.3
        }
    }
```
